# Bug Report Helper {#bug-report-helper}

AAO: Avatar Optimizerでバグを見つけた場合、報告していただけると助かります。  
バグを理解して修正するために、アバターや環境に関する情報が役に立ちます。

そのような情報を提供しやすくするため、AAO: Avatar Optimizer には Bug Report Helper ウィンドウが含まれています。  
このウィンドウはアバターと環境の情報をレポートファイルにまとめます。

このツールはライセンス違反やプライバシーの問題を避けるため、アバターのすべての情報を収集するのではなく、一般的な基本情報のみを収集する点にご注意ください。  
そのため、追加でアバターの詳細情報を求められる場合があります。

## 使い方 {#how-to-use}

Bug Report Helper ウィンドウはメニューの `Tools > Avatar Optimizer > Bug Report Helper` から開けます。

ウィンドウ上部には、バグを報告する対象のアバターを指定するフィールドがあります。  
AAO のバグに関連するアバターを設定してください。

ウィンドウ下部には 2 つのボタンがあります: `Save Bug Report` と `Copy Bug Report to Clipboard`。

`Save Bug Report` をクリックすると、バグレポートを圧縮（gz）ファイルとして保存するためのファイル保存ダイアログが開きます。これは効率的なため推奨される送信方法です。

`Copy Bug Report to Clipboard` をクリックすると、バグレポートのテキストがクリップボードにコピーされます。アプリケーションによっては貼り付けられないほど大きなテキストになる場合がある点にご注意ください。

どちらのボタンでもバグレポートファイルを作成できます。バグレポートを生成するために、Avatar Optimizer は一度アバターのビルドを行います。  
バグレポート作成時はファイルを添付するか、ファイル共有サービスにアップロードしてリンクを共有してください。

## レポート内容 {#report-contents}

バグレポートには以下の情報(一部)が含まれます:
- アバター情報
  - アバター名
  - アバターの Unity バージョン
  - GameObject とコンポーネントの一覧
  - 一部コンポーネントの設定（メッシュ情報を含む）
- 環境情報
  - Unity のバージョン
  - オペレーティングシステム
  - Avatar Optimizer のバージョン
  - 他にインストールされているパッケージのバージョン
- ビルド情報
  - ビルドプロセスのログ（警告やエラーを含む）
